# 🎓 UniSenai Mato Grosso - GRT.025.011

Bem-vindo à organização do GitHub da turma **GRT.025.011** do **UniSenai Mato Grosso**. 

Este espaço foi criado para centralizar, organizar e versionar nossos projetos acadêmicos, trabalhos práticos e laboratórios do curso de **Análise e Desenvolvimento de Sistemas**. Nosso objetivo é manter um portfólio colaborativo das nossas evoluções ao longo dos semestres.

---

## 🎯 Nosso Objetivo

- **Organização:** Manter todos os projetos acadêmicos em um único lugar, facilitando o acesso e a revisão.
- **Colaboração:** Permitir que os membros da equipe trabalhem juntos em trabalhos de grupo utilizando boas práticas de versionamento (Git/GitHub).
- **Portfólio:** Construir uma base sólida de código que sirva de vitrine para o nosso aprendizado em desenvolvimento de software, desde a base teórica até aplicações práticas.
